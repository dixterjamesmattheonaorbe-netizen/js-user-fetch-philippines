# js-user-fetch-philippines
A JavaScript web application that uses async/await to fetch user data from the JSONPlaceholder API. This project displays a directory of friends with their names, emails, and specific Philippine cities like Davao, Cebu, and Manila. Features include error handling with try...catch and a responsive CSS card layout.
# My Barkada Directory (PH Edition)

### Assignment Requirements:
- [x] Written in strictly JavaScript.
- [x] Uses `async` and `await` for API calls.
- [x] Fetches from `https://jsonplaceholder.typicode.com/users`.
- [x] Displays Name, Email, and City.
- [x] Uses `try...catch` for error handling.
